### Colors

## 🧩 Key Features

### ✨ Recent: User Preferences System (NEW!)
- **Pronoun Selection:** he/she/they + "Prefer not to say"
- **Tone Settings:** Warm, Professional, Concise
- **Accessibility Modes:** High Contrast, Reduced Motion, Larger Text
- **localStorage Persistence:** Saved across sessions
- **Real-time Updates:** Instant UI changes

### 🧭 Dual Navigation Mode
- **Tracking Mode:** Date-anchored, auto-unlocking cards
- **Exploring Mode:** Free browsing, all days accessible
- **Mode Switcher:** Toggle with confirmation modal
- **Keyboard Shortcuts:** Shift+T (Tracking), Shift+E (Exploring)

### 📊 28-Day Recovery System
- **2 Procedures:** Circumcision, TURP
- **29 Days Each:** Day 0 (surgery) → Day 28
- **BAUS-Compliant:** Clinical guidance from British Association
- **Dynamic Cards:** Normal experiences, forecast, red flags

## 🚀 Export Options

### Option 1: Anima Export (Recommended)
**Best for:** Figma-to-code workflow

1. Follow **ANIMA_EXPORT_GUIDE.md**
2. Export entire frame as React component
3. Download code package
4. Integrate with existing codebase

**Time:** 20-30 minutes

---

### Option 2: Direct Use (Code-First)
**Best for:** Developers using existing code

1. Use current codebase as-is
2. Install dependencies: `npm install`
3. Build: `npm run build`
4. Deploy: Vercel, Netlify, Cloudflare Pages

**Time:** 5 minutes

---

### Option 3: Figma Import First
**Best for:** Design-first workflow

1. Import text styles (Mode Select → "Text Styles")
2. Import variables (Mode Select → "Export Variables")
3. Set up Figma file with tokens
4. Then follow Option 1 (Anima Export)

**Time:** 1 hour (includes Figma setup)

## 🧪 Quick Test

### Test the User Preferences System
1. Start dev server: `npm run dev`
2. Navigate through: Splash → Welcome → **Personalise**
3. Select preferences (pronoun, tone, accessibility)
4. Click "Continue"
5. Navigate to Recovery screen
6. Verify:
   - Pronouns used correctly
   - Tone affects encouragement display
   - Accessibility modes apply instantly

### Test Dual Navigation Mode
1. From Mode Select, choose "Explore"
2. Select procedure, skip date
3. Navigate to Recovery screen
4. Try:
   - Click mode switcher (top-right)
   - Switch to "Tracking" mode
   - See confirmation modal
   - Confirm switch
   - Notice locked future days
