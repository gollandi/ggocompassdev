Mood Tracking Test Instructions

Testing the New Mood Slider Features:

1. Color Coding Test

  1. Go to: Recovery Page (Tracking Mode)
  2. Look for the "How confident do you feel about your recovery today?" slider
  3. Move the slider and observe:
    • Left side (1-3): Should show RED colors (uncertain)
    • Middle (4-7): Should show YELLOW colors (neutral)
    • Right side (8-10): Should show GREEN colors (certain)

2. Emergency CTA Test

  1. Move the slider to the left (1-3 range)
  2. You should see a red emergency box appear with:
    • "If you need immediate help:" message
    • Emergency contact instructions
    • Call clinical team, A&E, or 999 guidance

3. Mood Tracking Over Time

  1. Use the mood slider several times
  2. Look for the "Confidence Tracking" section
  3. It should show:
    • 7-day average
    • Trend indicators (improving/declining/stable)
    • History when expanded

4. Tracking Mode Only Test

  1. Switch to "Exploring" mode at the top
  2. The mood slider should DISAPPEAR
  3. Switch back to "Tracking" mode
  4. The mood slider should REAPPEAR

5. localStorage Test

  1. Set a mood level
  2. Refresh the page
  3. The mood slider should remember your last setting
  4. Check Developer Tools → Application → Local Storage → localhost:3000
  5. Look for "ggo_moodHistory" key with your mood data

Expected Behavior:

  • ✅ Color coding: Red (uncertain) → Yellow (neutral) → Green (certain)
  • ✅ Emergency CTA: Appears when mood is 1-3
  • ✅ Tracking mode only: Slider only visible in tracking mode
  • ✅ Persistence: Mood levels saved and tracked over time
  • ✅ History: 7-day tracking with trends
